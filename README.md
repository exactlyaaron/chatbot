# chatbot [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Chatbot to play YouTube videos

## Installation

```sh
$ npm install --save chatbot
```

## Usage

```js
var chatbot = require('chatbot');

chatbot('Rainbow');
```
